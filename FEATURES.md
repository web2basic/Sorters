# Sorters - Feature List

## Core Features

### ✅ Note Management
- Create notes with title and content (up to 10KB)
- Update existing notes
- Delete notes (owner only)
- Automatic version tracking
- Timestamp tracking (created/updated)

### ✅ Organization
- Folder system for organizing notes
- Tag system (up to 10 tags per note)
- Tag-based filtering and search

### ✅ Security & Privacy
- Optional encryption flag
- Owner-based access control
- Permission-based sharing

### ✅ Sharing & Collaboration
- Share notes with other users
- Read-only or write permissions
- Revoke sharing access
- Access control verification

### ✅ Blockchain Features
- Fully on-chain storage
- Immutable note history
- Transparent ownership
- No central server required

## Technical Features

### Smart Contract (Clarity)
- Type-safe contract language
- Gas-efficient operations
- Error handling with custom error codes
- Read-only functions for data queries

### Frontend (Next.js)
- Modern React with TypeScript
- Stacks wallet integration
- Responsive design
- Dark mode support
- Real-time transaction status
